Ferramentas necessárias

Para iniciar nossos estudos com JavaScript precisaremos de algumas poucas ferramentas. Todas elas são amplamente utilizadas e podem ser acessadas facilmente. Nada aqui será pago, pode ficar tranquilo(a). Precisaremos sempre de um bom editor de texto e um bom navegador Web para que possamos implementar e testar nossos projetos.

Aqui vale ressaltar que, em algumas ocasiões, criaremos projetos Javascript localmente, utilizando um editor de texto local. Isso quer dizer que você irá executar tudo diretamente no seu computador. Os projetos, nestes casos, ficarão armazenados no seu dispositivo.

Em outros casos, sempre que formos testar projetos menos complexos, utilizaremos algum editor de texto na nuvem (online). Existem diversas opções que nos possibilitam criar projetos Javascript de maneira bem rápida e intuitiva. É bem tentador fazer tudo dessa forma, mas, é importante que saibamos implementar projetos locais, criar pastas e arquivos todos por conta própria, pois, é assim que grande parte dos desenvolvedores procede.

Não se preocupe se, por enquanto, você não entendeu bem como são essas duas possibilidades. Assim que começarmos a programar, essas situações ficarão mais evidentes.

Editores de texto indicados​

  • Sublime Text 3
  • Visual Studio Code
  • Codepen (online)

Navegadores Web indicados​

  • Google Chrome
  • Microsoft Edge
  • Brave
  • Mozilla Firefox
